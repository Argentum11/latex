# Latex

## paper size

To change paper size, use

```tex
\documentclass[10pt, letterpaper]
```

- letterpaper
  - 8.5 x 11 inches
- a4paper

## macro

### \def and \setcommand

- \def
- \setcommand
  - checks whether or not the command already exists
  - allows you to define an optional argument
 
## comment

To comment a line of command, use %, for example

```tex
%$$x=\frac{1}{4}$$
```
